Non-Steam installation

Three steps, no installer

The archive already contains the correct mod folder. Existing local settings and generated translations are outside this folder and are not overwritten.

  1. 01

    Download

    Get the current ZIP from the button above.

  2. 02

    Extract

    Place the folder in RimWorld\Mods\RimWorldAutoTranslator

  3. 03

    Enable

    Install Harmony, load it first, then enable Auto Translation Framework.
