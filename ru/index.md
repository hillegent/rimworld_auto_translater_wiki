Auto Translation Framework — мод для RimWorld, который переводит другие активные моды, не изменяя их файлы из Мастерской. Он находит отсутствующую локализацию, повторно использует локальные и общие переводы, отправляет оставшийся текст выбранным переводчикам и безопасно применяет готовый результат во время игры там, где это допускает RimWorld.

**[Установить из Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3759370650)** | **[Открыть руководство](user-guide.html)**

## Что переводит мод

- **Keyed-строки**, используемые интерфейсом, предупреждениями, сообщениями и системами модов.
- **Поля DefInjected**: названия, описания, мысли, черты, предметы, здания, события, исследования и поддерживаемые пользовательские поля Def.
- **Некоторые захардкоженные строки C# UI**, которые перехватываются во время работы окон, кнопок, контекстных меню, команд, подсказок, строк осмотра, писем и сообщений.
- **Существующие переводы** из совместимых модов локализации и поддерживаемых кэшей других автопереводчиков.

Фреймворк хранит постоянную память переводов и создаёт XML-кэш для следующих запусков. Существующие человеческие переводы по умолчанию сохраняются и имеют приоритет.

## Основные возможности

- Автоматическое сканирование при запуске и фоновая очередь перевода.
- Определение целевого языка по текущему языку интерфейса RimWorld.
- Бесплатные web-переводчики, не требующие API-ключа в стандартной конфигурации.
- Дополнительная поддержка OpenAI, Claude, OpenRouter, DeepSeek, Gemini, DeepL, Microsoft, Google Cloud, LibreTranslate и OpenAI-совместимых сервисов.
- Режим нескольких провайдеров с настраиваемым параллелизмом и профилями.
- Включение, отключение, прогресс, повтор, применение и сброс отдельно для каждого мода.
- Полноэкранный редактор для поиска, исправления и повторного перевода отдельных строк.
- Опциональная общая библиотека с проверенной базой, авторскими пакетами, оценками и человеческими исправлениями.
- Проверка плейсхолдеров, форматирования и rich-text тегов перед принятием результата.
- Локальная диагностика, проверка здоровья провайдеров, задержки повторов и сетевой предохранитель.

## Быстрый старт

1. Установите [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077) и Auto Translation Framework из [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3759370650).
2. В списке модов разместите Harmony перед Auto Translation Framework.
3. Выберите нужный язык интерфейса RimWorld.
4. Откройте **Настройки > Настройки модов > Auto Translation Framework**.
5. На вкладке **Главное** оставьте включённым пункт **Использовать язык интерфейса RimWorld**.
6. На вкладке **Переводчики** оставьте **Google Translate Web** для самого простого варианта без ключа или настройте другой сервис.
7. Дождитесь окончания стартового сканирования. Вкладка **Моды** покажет прогресс и количество оставшихся строк для каждого активного мода.

Обычный рабочий процесс автоматизирован. Подробное описание настроек и решение частых проблем находятся в **[руководстве пользователя](user-guide.html)**.

## Как работает конвейер

1. Активные моды сканируются на поддерживаемые строки.
2. Сначала проверяются существующие человеческие переводы и локальная память.
3. Если общая библиотека включена, выполняется поиск готового проверенного перевода.
4. Отсутствующие записи отправляются выбранному провайдеру или пулу провайдеров.
5. Результат проверяется на сохранность плейсхолдеров, форматирования и rich-text тегов.
6. Готовые записи сохраняются локально, безопасно применяются во время игры и записываются в XML-кэш для следующих запусков.

Исходные файлы модов из Steam Workshop не изменяются.

## Общая библиотека переводов

Опциональная библиотека уменьшает количество повторной работы. Она может предоставлять проверенную базу и авторские пакеты, принимать человеческие исправления и повторно использовать готовые строки для совпадающих модов и языков.

Загрузка и отправка управляются раздельно. Можно оставить все переводы локальными, включить только загрузку либо явно активировать функции обмена. Отправленные материалы могут пройти автоматическую проверку перед публикацией в качестве доверенного общего пакета.

Информация о работе сервиса и финансировании находится на **[странице инфраструктуры](infrastructure-and-funding.html)**.

## Безопасность и ограничения

- Качество зависит от провайдера, исходного текста и статуса проверки библиотечного пакета.
- Перевод захардкоженного UI работает по принципу best effort, поскольку моды могут использовать собственные способы отрисовки.
- Для полного отображения некоторых новых DefInjected-переводов может потребоваться перезапуск RimWorld.
- Перевод области `Strings` пока выключен: эти файлы требуют отдельной безопасной обработки форматов.
- API и сетевые функции можно отключить; ключи нужны только тем провайдерам, которые их требуют.

Проект экспериментальный и активно развивается. Полное покрытие каждого мода и каждой динамически созданной строки не гарантируется.
